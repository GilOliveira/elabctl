# elabctl.sh

This script is used to manage an eLabFTW installation on a drop.

## Install

As root:

~~~bash
wget -qO- https://get.elabftw.net > /usr/bin/elabctl && chmod +x /usr/bin/elabctl
~~~

## Use

~~~bash
elabctl help
~~~
